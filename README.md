Golog 一款复刻自 [Tunalog](https://github.com/caris-events/tunalog) 的博客程序，在原项目基础上新增了如下特性：

- 支持渲染 Mermaid 图表，Latex 表达式
- 支持博文分类，随笔，时刻，日志
- 主题资源内嵌编译后的可执行文件产物里
- 默认主题重构

使用请前往 [releases](https://codeberg.org/wsh233/golog/releases) 下载编译好的二进制可执行文件。
