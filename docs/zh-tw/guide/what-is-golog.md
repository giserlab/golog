# 什麼是 Golog

Golog 是一款使用 Go 語言編寫的簡單、輕量部落格程式，復刻自 [Tunalog](https://github.com/caris-events/tunalog) 並在其基礎上進行了大量改進與重構。

## 專案定位

Golog 的目標是為個人使用者提供一個**開箱即用、單檔案部署、功能完備**的部落格解決方案。無需複雜的資料庫設定，無需繁瑣的依賴安裝，下載單個可執行檔案即可執行自己的部落格。

## 核心設計理念

- **簡單至上**：沒有複雜的設定項，啟動後透過精靈完成初始化即可開始寫作。
- **單檔案部署**：主題、模板、靜態資源全部內嵌在二進位檔案中，部署時只需一個檔案。
- **行動優先**：預設主題針對行動端進行了專門最佳化，確保讀者在任何裝置上都能獲得良好體驗。
- **內容為王**：支援 Mermaid 圖表、LaTeX 公式等豐富內容格式，讓技術部落格更具表現力。

## 適用場景

- 個人技術部落格
- 生活隨筆記錄
- 輕量級知識庫
- 專案文件站點
