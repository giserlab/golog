# 什么是 Golog

Golog 是一款使用 Go 语言编写的简单、轻量博客程序，复刻自 [Tunalog](https://github.com/caris-events/tunalog) 并在其基础上进行了大量改进与重构。

## 项目定位

Golog 的目标是为个人用户提供一个**开箱即用、单文件部署、功能完备**的博客解决方案。无需复杂的数据库配置，无需繁琐的依赖安装，下载单个可执行文件即可运行自己的博客。

## 核心设计理念

- **简单至上**：没有复杂的配置项，启动后通过向导完成初始化即可开始写作。
- **单文件部署**：主题、模板、静态资源全部内嵌在二进制文件中，部署时只需一个文件。
- **移动优先**：默认主题针对移动端进行了专门优化，确保读者在任何设备上都能获得良好体验。
- **内容为王**：支持 Mermaid 图表、LaTeX 公式等丰富内容格式，让技术博客更具表现力。

## 适用场景

- 个人技术博客
- 生活随笔记录
- 轻量级知识库
- 项目文档站点
