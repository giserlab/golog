# What is Golog

Golog is a simple, lightweight blog program written in Go, forked from [Tunalog](https://github.com/caris-events/tunalog) with significant improvements and refactoring.

## Project Positioning

Golog aims to provide individual users with an **out-of-the-box, single-file deployment, fully-featured** blogging solution. No complex database configuration, no tedious dependency installation — download a single executable and run your own blog.

## Core Design Philosophy

- **Simplicity First**: No complex configuration options. Complete initialization through the wizard after starting, then begin writing.
- **Single-File Deployment**: Themes, templates, and static assets are all embedded in the binary — only one file needed for deployment.
- **Mobile-First**: Default themes are specially optimized for mobile devices, ensuring readers get a great experience on any device.
- **Content is King**: Supports Mermaid diagrams, LaTeX formulas, and other rich content formats to make technical blogs more expressive.

## Use Cases

- Personal tech blog
- Life journaling
- Lightweight knowledge base
- Project documentation site
