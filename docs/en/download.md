# Download

Get the latest version of Golog through the following methods.

## Pre-built Binaries

Head to the GitHub Releases page to download the version for your system:

[Go to Releases →](https://github.com/WShihan/golog/releases)

### Supported Platforms

| OS      | Arch  | Filename                        |
| ------- | ----- | ------------------------------- |
| macOS   | ARM64 | `golog_darwin_arm64/golog`      |
| Linux   | AMD64 | `golog_linux_amd64/golog`       |
| Windows | AMD64 | `golog_windows_amd64/golog.exe` |

## Source Download

```bash
git clone https://github.com/WShihan/golog.git
```

## Release History

For full changelog, visit the [Releases](https://github.com/WShihan/golog/releases) page.
